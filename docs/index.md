# Документация проекта

`cvd-risk-prediction-mlops` — учебный ML/MLOps-проект по теме прогнозирования риска сердечно-сосудистых осложнений на основе ретроспективных данных пациентов.

В рамках ЛР5 в проект была добавлена базовая система документации на `MkDocs`. Она нужна для того, чтобы хранить в одном месте описание структуры репозитория, назначения ключевых модулей и примеры автодокументирования Python-кода через `docstring`.

## Что входит в документацию

- краткое описание проекта и его предметной области;
- обзор структуры репозитория;
- описание основных модулей из каталога `src/`;
- заготовка для автоматической генерации страниц по `docstring` через `mkdocstrings`.

## Практическая польза

Документация позволяет быстрее ориентироваться в проекте, упрощает сопровождение кода и делает структуру ML-системы более понятной для разработчика, преподавателя и нового участника проекта.

Далее в документации можно расширять разделы по подготовке данных, обучению моделей, экспериментам в `MLflow` и сценариям инференса.
