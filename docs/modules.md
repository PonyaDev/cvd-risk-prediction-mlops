# Основные модули

В проекте выделены несколько ключевых модулей, связанных с обучением и сопровождением модели прогнозирования риска сердечно-сосудистых осложнений.

## Краткое описание

- `src.main` — минимальная точка входа проекта.
- `src.data.demo_dataset` — подготовка безопасного демонстрационного набора данных.
- `src.models.train_experiment` — запуск обучения и логирования эксперимента в `MLflow`.
- `src.models.evaluation` — расчёт метрик качества модели.
- `src.models.register_best_model` — регистрация лучшей версии модели в `MLflow Model Registry`.
- `src.models.champion_challenger` — имитация сценария `Champion/Challenger`.
- `src.utils.mlflow_config` — локальная конфигурация трекинга и хранения артефактов `MLflow`.
